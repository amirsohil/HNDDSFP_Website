Points System

The following system based on the official FPL points system was used for comparing the players chosen by the created models with the actual players included in the respective PFA TOTY chosen via the human-based voting system. Here, the higher a given player's points are, the better the player in the considered season.

Accordingly, points were awarded as follows:

Event Points Awarded
For each set of 90 minutes played 2
For each goal scored by a goalkeeper or defender 6
For each goal scored by a midfielder 5
For each goal scored by a forward 4
For each assist for a goal 3
For a clean sheet by a goalkeeper 4
For every 3 tackles or interceptions by a defender 1
For every 5 tackles or interceptions by a midfielder 1
For every 3 shots saved by a goalkeeper 1
For each penalty save 5
For each penalty miss -2
For every 2 goals conceded by a goalkeeper -1
For each yellow card received -1
For each red card received -3
For each own goal scored -2
